# 동적시드 기능 가이드

## 개요
동적시드는 시장 하락 또는 T값 기준 도달 시 자동으로 시드를 증액하는 기능입니다.

## 설정 항목

| 항목 | 설명 | 기본값 |
|------|------|--------|
| 동적시드 스위치 | OFF: 기존과 동일 (동적시드 미사용)<br>ON: 동적시드 활성화 | OFF |
| 동적시드 Max | 증액 가능한 최대 시드 | - |
| 증액배수 | 조건 충족 시 증액 비율 (Max 초과 불가) | 30% |
| 하락률 기준 | 전일대비 N% 하락 시 증액 | 3% |
| T임계점 | T값이 max_tier의 N% 도달 시 증액 | 30% |

## 증액 조건
다음 중 **하나라도** 충족하면 증액:
1. **하락률 조건**: 전일대비 하락률 ≥ 하락률 기준
2. **T값 조건**: 현재 T값 ≥ max_tier × T임계점

> T임계점 100% 설정 시 → T값 조건 비활성화 (하락률만 검사)

## 동일 심볼 규칙
- 같은 심볼의 봇이 여러 개일 경우, **하루에 1개만 증액**
- 시드가 작은 봇 우선
- 예: TQQQ_1(시드 \$500), TQQQ_2(시드 \$1000) → TQQQ_1만 증액, TQQQ_2는 패스
