# 동적 시드 (Dynamic Seed) 기능 설명
## 기능 요약
주가 하락 시 자동으로 시드를 증가시켜 저점 매수 기회를 늘리는 기능

---

## 설정 방법

### 기존 방식
- Seed: $500 (고정)

### 동적 시드 사용 시
- Seed: $50 (작게 시작)
- 동적시드Max: $500 (기존 시드값 입력)

→ 평소엔 $50으로 운영, 하락장에서 최대 $500까지 자동 증가

> 동적시드Max = 0 이면 기능 비활성화

---

## 티커별 하락률 기준

| 티커 | 하락률 인터벌 | 비고 |
|------|--------------|------|
| TQQQ | 3% | QQQ 3배 레버리지 |
| 그 외 | 5% | SOXL 등 |

> TQQQ는 변동성이 상대적으로 낮아 인터벌을 좁게 설정

---

## 작동 원리

두 단계를 독립 계산 후, **둘 중 큰 값**이 적용됩니다.

### 1단계: 전일대비 하락

어제보다 일정% 이상 떨어지면 **현재 시드 × 1.2배**

| 티커 | 조건 |
|------|------|
| TQQQ | 3% 이상 하락 시 |
| SOXL | 5% 이상 하락 시 |

> 현재 시드 기준이라 연속 하락 시 누적 증가

### 2단계: 90일 고점대비 하락

고점 대비 하락률에 따라 **동적시드Max의 일정 비율**

| TQQQ (3% 단위) | SOXL (5% 단위) |
|----------------|----------------|
| -3% → Max의 10% | -5% → Max의 10% |
| -6% → Max의 20% | -10% → Max의 20% |
| -9% → Max의 30% | -15% → Max의 30% |
| -12% → Max의 40% | -20% → Max의 40% |
| -15% → Max의 50% | -25% → Max의 50% |
| ... | ... |
| -30% → Max의 100% | -50% → Max의 100% |

---

## 예시

### 예시 1: 2단계 적용

**설정**
- Seed: $100
- 동적시드Max: $1000

**상황**: 고점대비 -15% 하락

| 단계 | 계산 | 결과 |
|------|------|------|
| 1단계 | 전일 변동 없음 | 해당없음 |
| 2단계 | Max × 50% | $500 |
| **최종** | | **$100 → $500** |

### 예시 2: 1단계 누적

**설정**
- Seed: $100
- 동적시드Max: $1000

**상황**: 매일 전일대비 -5% 하락

| 일차 | 계산 | 시드 |
|------|------|------|
| 1일차 | $100 × 1.2 | $120 |
| 2일차 | $120 × 1.2 | $144 |
| 3일차 | $144 × 1.2 | $173 |
| ... | 연속 하락 시 | $1000 도달 가능 |

---

## 주의사항

- 시드가 동적시드Max에 도달하면 더 이상 증가하지 않음
- 동일한 상황 유지 시 시드 변동 없음
- 시드 증가 시 텔레그램 알림 발송
