# 봇 리뉴얼 사용 가이드

## 봇 리뉴얼이란?

시장 상황에 따라 **모든 봇의 투자 설정을 한 번에 조정**하는 기능입니다.

## 시장 레벨

| 레벨 | 이름 | 언제 사용? |
|------|------|-----------|
| 🛡️ Lv.0 | 수비적 | 약세장, 불확실한 시장 |
| ⚖️ Lv.1 | 중립 | 보합장, 일반적인 상황 |
| ⚔️ Lv.2 | 공격적 | 강세장 초기 |
| 🔥 Lv.3 | 매우공격적 | 강한 강세장 |

## 사용 방법

1. **봇 정보 페이지**에서 "🔄 봇 리뉴얼" 버튼 클릭
2. **시장 레벨** 선택 (예: Lv.2 공격적)
3. **미리보기** 확인:
   - 예산 배분 (현금 비중, 투자 금액)
   - 각 봇의 변경 사항 (1회 시드, 수익률, Max티어 등)
4. **리뉴얼 적용** 버튼 클릭

## 총 예산 변경 (선택사항)

추가 자금을 투입하거나 예산을 조정할 때:

1. "총 예산" 입력란에 금액 입력 (예: `90000`)
2. 1초 후 자동으로 미리보기 갱신
3. 시장 레벨 선택 후 리뉴얼 적용

**중요**: 입력한 총 예산은 저장되어 다음에도 유지됩니다.

## 수동 설정 모드

개별 봇을 직접 수정하면 자동으로 `⚙️ 수동설정` 모드로 전환됩니다.

**복귀 방법**:
1. 봇 리뉴얼 모달 열기
2. 원하는 시장 레벨 선택
3. 리뉴얼 적용

## 봇 현황 카드

```
🛡️ Lv.1 중립    총액 $54,000    1회 $10,000
```

- **Lv.1 중립**: 현재 시장 레벨 (또는 수동설정)
- **총액**: 모든 봇의 최대 투자금 합계
- **1회**: 모든 봇의 1회 시드 합계

## 주의사항 ⚠️

1. **되돌릴 수 없음** - 미리보기를 꼼꼼히 확인하세요!
2. **모든 봇 활성화** - 꺼져있던 봇도 자동으로 켜집니다
3. **동적 시드 초기화** - 모든 봇의 동적 시드가 꺼집니다
4. **즉시 매매 발생 가능** - 새로운 1회 시드가 현재 설정보다 낮아지면 리뉴얼 직후 바로 매도가 일어날 수 있습니다

## FAQ

**Q. 총 예산을 입력했는데 적용이 안 돼요**
A. 입력 후 1초를 기다려주세요 (자동 갱신)

**Q. 수동설정 모드에서 벗어날 수 없어요**
A. 봇 리뉴얼에서 시장 레벨을 선택하고 적용하세요

**Q. 리뉴얼 전 설정으로 되돌릴 수 있나요?**
A. 불가능합니다. 미리보기를 충분히 확인 후 신중하게 적용하세요

## 실전 예시

### 강세장 진입
1. 봇 리뉴얼 → "⚔️ Lv.2 공격적" 선택
2. 미리보기: 투자 금액 $25,000 → $35,000
3. 적용 → 모든 봇이 공격적으로 투자

### 추가 자금 투입
1. 총 예산에 `90000` 입력 (기존 $50,000 → $90,000)
2. 미리보기: 각 봇의 1회 시드 증가 확인
3. "⚖️ Lv.1 중립" 선택 후 적용
