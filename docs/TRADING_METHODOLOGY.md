# 티어 기반 분할 매매 전략

## 개요

이 전략은 **"포지션 크기에 따라 익절/손절 기준을 자동 조절하자"**는 아이디어에서 출발한다.

**핵심 철학:**
- 한 번에 올인하지 않는다 → 단계적으로 분할 진입한다
- 최대 투자 한도를 정해둔다 → 무한 추가 매수 방지
- 포지션이 커질수록 청산 기준을 낮춘다 → 리스크 관리

---

## 1. 티어(Tier) - 전체 투자 중 얼마나 진입했는가?

### 왜 티어가 필요한가?

한 종목에 투자할 때, 처음부터 전액을 투입하는 건 리스크가 크다. 그래서 **최대 투자금을 정해두고, 조건에 맞을 때마다 조금씩 진입**한다.

이때 **"전체 중 얼마나 진입했는가?"**를 나타내는 것이 **티어(T)**다.

### 티어 계산법

```
티어 = 지금까지 투자한 금액 / 1회 투자 금액
```

**예시: 1회 투자금 $1,000, 최대 10회까지 투자 가능 (= 최대 $10,000)**

| 상황 | 누적 투자 | 티어 | 의미 |
|------|----------|------|------|
| 첫 매수 | $1,000 | T=1 | 10% 진입 |
| 추가 매수 | $2,000 | T=2 | 20% 진입 |
| 추가 매수 | $3,000 | T=3 | 30% 진입 |
| ... | ... | ... | ... |
| 마지막 매수 | $10,000 | T=10 | 100% 진입 (풀 포지션) |

### 티어는 수익/손실과 무관하다

```
케이스 1: 주가 하락 중 → 싸게 사서 평단가 낮추기 (물타기)
케이스 2: 주가 상승 중 → 추세 따라 추가 진입 (불타기)

둘 다 티어가 올라간다. 티어는 "방향"이 아니라 "진입량"을 측정한다.
```

### 티어가 높아지면 무슨 일이?

```
T 낮음 (1~3)  →  포지션 작음. 추가 매수 여력 충분. 여유로움.
T 중간 (4~6)  →  포지션 중간. 신중하게 추가 진입.
T 높음 (7~9)  →  포지션 큼. 추가 매수 여력 거의 없음. 청산 고려.
T 최대 (10)   →  풀 포지션. 더 이상 추가 불가. 청산 타이밍 탐색.
```

**핵심:** 티어가 올라갈수록 "언제 나갈 것인가"에 집중해야 한다.

## 2. 동적 익절/손절 지점 - 포지션 크기에 따라 기준이 바뀐다

### 고정 익절의 문제점

보통 "평단가 대비 +10%면 익절"처럼 고정된 기준을 사용한다.

하지만 이 방식의 문제:
- 포지션이 커진 상태에서 +10%까지 기다리면 리스크가 큼
- 반대로 포지션이 작을 때 +10%에 팔면 기회비용 손실

### 동적 지점의 아이디어

**포지션 크기(티어)에 따라 청산 기준을 자동 조절한다.**

```
포지션 작을 때 (T 낮음): "여유 있으니 +20%까지 기다리자" → 높은 목표
포지션 클 때 (T 높음): "리스크 크니 본전이라도 나가자" → 낮은 목표
```

### 구체적인 작동 방식

**설정값:**
- 기준폭: 20% (목표 지점의 최대 범위)
- 전환점: 50% (1/2 전략의 경우)

**티어별 목표 지점:**

```
[높은 목표]                    [낮은 목표]
   +20% ─────┐
             │
   +10% ─────┼─── T=25%  (포지션 25%)
             │
    0% ──────┼─── T=50%  (포지션 50%, 전환점)
             │
   -10% ─────┼─── T=75%  (포지션 75%)
             │
   -20% ─────┘─── T=100% (풀 포지션)
```

**해석:**
- T=1 (10% 진입): 목표 +18% → "조금밖에 안 샀으니 크게 먹자"
- T=5 (50% 진입): 목표 ±0% → "반쯤 샀으니 본전이면 정리 고려"
- T=9 (90% 진입): 목표 -18% → "거의 다 샀으니 손해봐도 정리"

### 왜 이렇게 하는가?

```
포지션 작을 때:
  "투자금이 적으니 변동성 감당 가능하다.
   천천히 기다려서 큰 수익을 노리자."

포지션 클 때:
  "투자금이 크니 변동성이 곧 큰 손실이다.
   욕심 줄이고 빨리 청산하자."
```

### 전환점 전략 비교

| 전략 | 전환점 | 특징 |
|------|--------|------|
| 1/2 전략 | 50% | 익절/손절 영역이 균등. 밸런스형. |
| 2/3 전략 | 67% | 높은 목표 영역이 더 넓음. 수익 기회↑, 빠른 손절 필요. |

## 3. 매수 조건 - 언제 추가 매수를 할 것인가?

### 기본 원칙

매일 무조건 사는 게 아니다. **"지금이 추가 진입하기 좋은 타이밍인가?"**를 판단해야 한다.

### 매수 조건 2가지

**조건 1: 평단가보다 낮은가?**
```
현재가 < 내 평단가  →  추가 매수하면 평단가가 낮아진다 (물타기 효과)
```

**조건 2: T 기반 동적 지점보다 낮은가?**
```
현재가 < 동적 지점가  →  "이 가격이면 살 만하다"고 판단
```

**동적 지점 = 티어(T)에 의해 계산되는 값**

```
┌─────────────────────────────────────────────────┐
│  동적 지점 = 평단가 × (1 + T 기반 퍼센트)        │
│                                                 │
│  T 기반 퍼센트는 2번 섹션의 공식으로 계산됨       │
└─────────────────────────────────────────────────┘
```

예시 (기준폭 20%, 1/2 전략):

| 티어 | T 기반 퍼센트 | 동적 지점 | 매수 조건 |
|------|--------------|----------|----------|
| T=1 | +18% | 평단가 × 1.18 | 현재가 < 평단가+18%면 매수 (불타기 가능) |
| T=5 | ±0% | 평단가 × 1.00 | 현재가 < 평단가면 매수 |
| T=9 | -18% | 평단가 × 0.82 | 현재가 < 평단가-18%면 매수 (신중) |

### 조건 조합에 따른 매수 비율

두 조건을 모두 켜놓았을 때:

| 평단가 조건 | 목표지점 조건 | 결과 |
|-------------|--------------|------|
| O | O | 100% 매수 (2/2 충족) |
| O | X | 50% 매수 (1/2 충족) |
| X | O | 50% 매수 (1/2 충족) |
| X | X | 매수 안함 (0/2 충족) |

**왜 비율로 매수하는가?**

"애매할 때는 절반만 투입"하는 전략이다. 확신이 없을 때 올인하지 않는다.

### 급락장 대응 - 빅드랍 감지

전일 대비 급락이 발생하면, 평소보다 더 적극적으로 매수한다.

```
전일 종가: $100
오늘 현재가: $94 (6% 하락)

→ "급락이니까 시드를 30~50% 더 투입하자"
```

| 하락폭 | 추가 투입 | 이유 |
|--------|----------|------|
| 소폭 하락 | 기본 시드 | 평소대로 |
| 중간 하락 | +30% | 기회일 수 있음 |
| 큰 하락 | +40% | 좋은 매수 기회 |
| 급락 | +50% | 놓치면 안 되는 기회 |

**주의:** 급락이 반등 기회일 수도 있지만, 더 떨어질 수도 있다. 그래서 최대 티어 제한이 있는 것이다.

## 4. 매도 조건 - 언제 팔 것인가?

### 두 가지 매도 기준

**기준 1: 고정 익절가 → 돌파 시 3/4 (75%) 매도**
```
고정 익절가 = 평단가 × (1 + 익절률)

예: 평단가 $100, 익절률 5%
→ 고정 익절가 = $105
→ 현재가 > $105 → 보유량의 75% 매도
```

**기준 2: T 기반 동적 지점가 → 돌파 시 1/4 (25%) 매도**
```
동적 지점가 = 평단가 × (1 + T 기반 퍼센트)

예: T=3, 기준폭 20%, 1/2 전략
→ T 기반 퍼센트 = +12%
→ 동적 지점가 = $100 × 1.12 = $112
→ 현재가 > $112 → 보유량의 25% 매도
```

### 조건 조합에 따른 매도량

| 고정 익절가 돌파 | 동적 지점 돌파 | 매도량 | 설명 |
|-----------------|---------------|--------|------|
| O | O | **100%** (4/4) | 둘 다 충족 → 전량 익절 |
| O | X | **75%** (3/4) | 고정만 충족 → 3쿼터 익절 |
| X | O | **25%** (1/4) | 동적만 충족 → 1쿼터 익절 |
| X | X | **0%** | 미충족 → 홀딩 |

### 왜 이런 비율인가?

```
고정 익절가 = 내가 설정한 "이 정도면 충분하다"는 기준
  → 더 확실한 신호 → 큰 비중(75%)으로 청산

동적 지점가 = 시스템이 T에 따라 계산한 기준
  → 상황에 따른 신호 → 작은 비중(25%)으로 청산
```

부분 매도는 **"확신의 정도에 따라 청산 비율을 조절"**하는 전략이다.

### 강제 청산 (손절)

**T ≥ 최대티어 - 1 이면:**
```
"거의 풀 포지션이다. 더 이상 추가 매수 여력이 없다."
"여기서 더 떨어지면 손실이 너무 커진다."
→ 손해를 감수하고 1/4 (25%) 손절
```

| 상황 | 매도량 | 설명 |
|------|--------|------|
| T=9, 최대=10 | **25%** | 1쿼터 손절로 포지션 축소 |
| 이후에도 조건 유지 | **25%씩** | 반복해서 포지션 줄임 |

**왜 전량 손절이 아닌가?**
```
급락 후 반등할 수도 있다.
1/4씩 손절하면:
  - 반등 시 → 나머지 75%로 회복 기회
  - 추가 하락 시 → 다음 날 또 1/4 손절로 리스크 감소
```

이것이 **점진적 손절** 전략이다.

## 5. 부가 기능

### 반복리
부분 매도로 얻은 수익금을 다음 매수에 추가 투입하여 복리 효과를 낸다.

### TWAP 분할 주문
한 번에 전량 거래하지 않고, 여러 번에 나눠서 체결하여 가격 충격을 최소화한다.

### 장부거래
같은 종목에 매수/매도 주문이 동시에 있으면, 실제 거래 없이 내부에서 상쇄 처리하여 수수료를 절감한다.
